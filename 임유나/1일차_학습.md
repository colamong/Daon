## Let
```js
let title = "학습 보고서"

// 변수 재할당 가능
title = "학습 보고 일지";
```

## Const
```js
const title = "학습 보고서"

// 변수 재할당 불가능
// title = "학습 보고 일지" (error)

// 변수 내부 속성을 변경 가능
const work_list = {
  title :  "학습 보고서"
};

work_list.title = "학습 보고 일지";

// 따라서, 특정 상황에 따라 내부 속성도 변경하지 못하도록 설정 할 수 있음.
```

## Var
- var는 변수가 변하는 것을 막을 수 없지만, 기존부터 써왔던 것이기에 이를 변경하기엔 부담이 컸음
- 따라서, const와 let이 생김
- 그렇기 때문에 **var를 쓰는 것은 지양**해야함.
